# 🚀 DIO Evolution - Solução de Desafios Iniciais de JavaScript

Este projeto foi desenvolvido como parte da minha jornada na formação **JavaScript Developer da DIO (Digital Innovation One)**. Ele reúne a resolução de três desafios lógicos clássicos, apresentados em uma interface web moderna e intuitiva.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização personalizada com tema Dark Mode.
* **JavaScript (ES6+):** Lógica de programação e manipulação dinâmica do DOM.
* **GitHub Pages:** Hospedagem e deploy contínuo.

---

## 🧠 Desafios Implementados

A aplicação conta com três módulos interativos:

1.  **🦆 Patinhos:** Implementação da lógica baseada na famosa música infantil, trabalhando com contagem e retorno de valores.
2.  **📊 Média de Notas:** Um calculador que recebe três notas, valida se os campos foram preenchidos e retorna a média aritmética formatada.
3.  **🔢 Dobro:** Um conversor simples que pega qualquer entrada numérica e exibe o dobro do valor instantaneamente.

---

## ✨ Diferenciais do Projeto

* **Validação de Inputs:** O código verifica se o usuário deixou campos em branco antes de processar a lógica, garantindo uma melhor UX (Experiência do Usuário).
* **Design Moderno:** Interface inspirada em dashboards de tecnologia, utilizando cores sóbrias e tipografia limpa.
* **Código Organizado:** Funções bem definidas e separadas por responsabilidade, facilitando a leitura e futuros upgrades.
