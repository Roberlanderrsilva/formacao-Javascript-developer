## 🧠 Desafios Implementados

A aplicação conta com três módulos interativos[span_13](start_span)[span_13](end_span)[span_14](start_span)[span_14](end_span):

1.  **🦆 Patinhos:** Implementação da lógica baseada na famosa música infantil, trabalhando com contagem e retorno de valores[span_15](start_span)[span_15](end_span)[span_16](start_span)[span_16](end_span).
